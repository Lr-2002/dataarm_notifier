# 更新日志

本项目遵循[语义化版本控制](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-12-15

### 新增
- 初始版本发布
- 支持控制三种颜色的灯（红、绿、蓝）
- 支持PWM调光（0-100%亮度控制）
- 支持颜色轮换功能（默认2秒间隔）
- 提供Socket API接口
- 基于Modbus RTU协议的串口通讯
- 支持CRC16校验
- 提供命令行工具
- 完整的测试套件
- 详细的文档

### 功能特性
- 独立控制红灯、绿灯、蓝灯
- 按Enter键开始/停止颜色轮换
- 轮换顺序：红 → 绿 → 蓝 → 红...
- 多线程实现，响应迅速
- 支持多客户端Socket连接
- 完整的JSON响应格式

### 技术规格
- 串口参数：4800波特率，8位，无校验，2停止位
- 通讯协议：Modbus RTU (0x06写寄存器)
- 设备路径：`/dev/cu.usbserial-1330` (macOS)
- 亮度范围：0-1999 (PWM值)

### 依赖
- pyserial >= 3.5

## 版本号规则

- **MAJOR**: 不兼容的API变更
- **MINOR**: 向后兼容的功能性变更
- **PATCH**: 向后兼容的问题修正

## 更新类型

- `新增`: 新功能
- `改进`: 现有功能的改进
- `修复`: Bug修复
- `废弃`: 即将删除的功能
- `移除`: 已删除的功能
- `安全`: 安全相关修复
